# Snapchat Clone Application

## Overview
The Snapchat Clone Application is designed to mimic the core functionalities of the popular multimedia messaging app, Snapchat. This project serves as an educational tool for developers looking to understand the architecture and important features of a social media application.

## Features
- **User Registration and Authentication:** Users can create accounts and log in securely.
- **Snaps:** Users can take photos and videos that disappear after being viewed.
- **Stories:** Users can post snippets of their day, which are viewable for 24 hours.
- **Chat Functionality:** Users can send text messages, photos, and videos to each other.
- **Filters and Lenses:** Similar to Snapchat, users can apply fun filters and lenses to their snaps.
- **User Profiles:** Each user has a unique profile showcasing their snaps and stories.

## Tech Stack
- **Frontend:** React Native for cross-platform mobile development.
- **Backend:** Node.js with Express for handling requests and managing the application logic.
- **Database:** MongoDB for storing user data and media files.
- **Cloud Storage:** AWS S3 for storing images and videos.

## Conclusion
The Snapchat Clone Application serves as a comprehensive project to learn about mobile app development, cloud services, and the intricate features of modern social media applications.
